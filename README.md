<h1>Company Brochure Generator</h1>

<p>
The <strong>Company Brochure Generator</strong> is a web-based tool that transforms any company website into a professional, concise, and visually appealing brochure in seconds. By leveraging advanced AI, this tool extracts key information from the company's website and relevant internal links, helping businesses, marketers, and individuals quickly create polished brochures without manual effort.
</p>

<hr>

<h2>Problem it Solves</h2>

<p>
Creating a professional company brochure is time-consuming and often requires marketing expertise. Businesses need to manually collect website content, product information, company achievements, and career details, then format it in an appealing way. The <strong>Company Brochure Generator</strong> automates this process by:
</p>
<ul>
  <li>Scraping the main website content.</li>
  <li>Identifying relevant internal links (product pages, services, case studies, testimonials).</li>
  <li>Extracting key information including mission, vision, values, culture, and career opportunities.</li>
  <li>Generating a polished brochure in markdown format, ready for sharing or printing.</li>
</ul>

<hr>

<h2>Key Features</h2>
<ul>
  <li>Professional brochure generation from any company website.</li>
  <li>Automatic content extraction and summarization.</li>
  <li>Incorporates company overview, products & services, culture, achievements, and career info.</li>
  <li>Generates brochure in attractive markdown format with easy download or copy options.</li>
  <li>User-friendly interface built with Streamlit for quick and seamless use.</li>
</ul>

<hr>

<h2>How It Works</h2>
<ol>
  <li>Enter the company's name and website URL.</li>
  <li>Click "Generate Brochure".</li>
  <li>The system analyzes the website and relevant internal links.</li>
  <li>Receive a concise, professional brochure ready for download or sharing.</li>
</ol>
